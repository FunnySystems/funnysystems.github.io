隱藏路徑

一台電腦中勒索病毒，不可怕；整個企業網路中勒索病毒，才是癱瘓企業營運能力的關鍵。

勒索病毒往往利用企業不設防的 隱藏路徑，達成感染的效果。

勒索病毒、資料外洩、敵國 APT，這些威脅都可以歸納為 隱藏路徑 的問題。

解決方案

RTP (Red Team Platform 威脅情境模擬平台)

RTP 是 Funny Systems (法泥系統) 所研發的威脅情境模擬系統，此系統相當於企業資訊安全的 #善意假想敵，可以模擬 勒索軟體 / 資料外洩 / 後門常駐 / 網路間諜 等行為; 以戰代訓，逐次提升企業防守實力。

RTP 在工作模式上，包含 匿蹤作戰 / 廣域偵蒐 / 持久作戰 / 閃電作戰 / 遠距打擊 / 敵後作戰 / 精準狙擊 等模式，以符合真實的攻擊想定。

AutoTTP (自動化戰術決策引擎)

RTP 的核心技術是，AutoTTP (自動化戰術決策引擎)，是作戰機器人的大腦，在整個假想演練過程中，AutoTTP 會不斷尋找企業環境的戰術破口，並評估其可利用性，自動組合各種奇技淫巧。

想想看，當 RTP 擁有超過 100 種以上戰術能力，作戰機器人每擴散到一台新的企業電腦節點，AutoTTP 就會挖掘到新的不同資源，並與作戰中心同步戰場情報，而作戰中心的圖形化介面的可視性，則可讓企業防守方順著時間軸，讓整個企業 IT 環境的可攻擊隱藏路徑一一浮現。

RTP 小組的其他作戰機器人，會與作戰中心共享這些情資，依照各自的網路空間地理環境，獲取到新的網段與目標節點，自動化地持續反復，直到達成想定的作戰任務。
